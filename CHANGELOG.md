# Changelog

## 0.1.0 - 2026-06-21

- Added workbook rendering for placeholders, helpers, row loops, column loops, grids, blocks, images, formulas, merge/style cloning, and multi-sheet workbooks.
- Added null-safe expression evaluation with default values and no `eval` or `Function` constructor.
- Added render limits for templates, worksheets, rows, columns, operations, and image payloads.
- Added secure default asset resolution for relative image paths, base64 images, and buffers.
- Added production documentation, Typedoc output, benchmarks, and a production showcase template with sample JSON data.
